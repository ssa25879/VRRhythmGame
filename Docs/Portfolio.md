# VR 리듬 액션 게임 포트폴리오

## 게임 개요

본 프로젝트는 Unity 기반 VR 리듬 액션 게임이다.

플레이어는 양손 컨트롤러를 세이버처럼 사용하여 음악과 노트맵 타이밍에 맞춰 날아오는 노트를 베고, 노트의 색상과 방향에 맞게 입력하여 점수와 콤보를 획득한다.

## 시연 영상

- 테스트 플레이 영상 일부공개 링크: https://youtu.be/3g2RF_wvLGw

## 담당 역할

본 프로젝트는 개인 프로젝트로 진행하였으며, Unity 프로젝트 환경 구성, VR 입력 처리, 세이버 판정, 노트 생성 로직, UI 구성, 테스트 및 밸런스 조정을 직접 수행하였다.

기능 구현과 프로젝트 정리 과정에서는 Claude Code와 Codex를 활용하여 개발 속도를 높였다.

- Unity 기반 VR 프로젝트 환경 구성
- Meta Quest 3S 테스트를 고려한 XR Interaction Toolkit 설정
- OpenXR 기반 VR 실행 환경 구성
- Intro 씬과 Game 씬 구성
- Intro 씬에서 스테이지 선택 UI 구현
- 컨트롤러 Ray를 이용한 VR UI 상호작용 기능 구현
- Game 씬에서 양손 세이버 오브젝트 구성
- 노트 생성 및 이동 로직 구현
- Beat Sage 자동 생성 노트맵 적용
- 노트 색상에 따른 좌/우 세이버 판정 구현
- 노트 방향에 따른 성공/실패 판정 구조 구현
- Score, Combo, HP, Miss 기능 구현
- Hit 성공 시 Combo 기반 HP 회복 기능 구현
- HP 상태를 확인할 수 있는 수직 HP Bar HUD 구현
- 곡 종료 또는 HP 0 도달 시 게임 종료 처리
- 게임 종료 후 결과 UI 출력 및 Intro 씬 복귀 기능 구현

- 작업 로그, 작업 목록, 프로젝트 지침 문서 관리

## 구현 기능

### 스테이지 선택 기능

Intro 씬에서 스테이지를 선택하고 Game 씬으로 이동할 수 있도록 구성하였다.

### VR 컨트롤러 입력 기능

컨트롤러 Ray를 이용하여 UI 버튼을 선택할 수 있도록 구성하였다.

### 세이버 판정 기능

좌우 컨트롤러에 각각 세이버를 배치하고, 노트의 색상과 방향에 맞게 휘둘렀을 때 성공 판정이 발생하도록 구현하였다.

### Beat Sage 노트맵 적용

Beat Sage에서 자동 생성한 노트맵 데이터를 프로젝트에 적용하여, 곡의 흐름에 맞춰 노트가 생성되도록 구성하였다.

### 점수 및 콤보 기능

노트 판정 결과에 따라 점수와 콤보가 증가하도록 하였으며, 콤보가 유지될수록 점수 획득에 의미가 생기도록 구조를 설계하였다.

### HP 및 Miss 기능

노트를 놓치거나 잘못 판정된 경우 HP가 감소하도록 구성하였다. Hit 성공 시 Combo 수에 따라 HP가 일부 회복되도록 하여 플레이 지속성을 높였다.

HP는 숫자뿐 아니라 수직 HP Bar로 표시하여 플레이 중 상태를 쉽게 확인할 수 있도록 하였다.

### 게임 종료 기능

HP가 0이 되거나 곡이 종료되고 모든 노트 처리가 끝나면 게임이 종료되며, 결과 UI가 표시되도록 구성하였다.

결과 확인 후 Intro 씬으로 돌아갈 수 있는 흐름을 구현하였다.

## 핵심 기술 포인트

- XR Interaction Toolkit을 활용한 VR 컨트롤러 입력 처리
- OpenXR 기반 Meta Quest 3S 테스트 환경 구성
- 좌우 컨트롤러 색상 기반 세이버 판정 구현
- 노트 방향과 세이버 이동 방향을 비교하는 판정 구조 구현
- Beat Sage 자동 생성 노트맵 데이터를 활용한 스테이지 구성
- HP, Combo, Score, Miss를 포함한 기본 게임 루프 구현

## 개발 중 개선한 문제

초기 테스트에서는 세이버의 시각적 위치와 실제 판정 위치가 어긋나는 문제가 있었다. 이를 해결하기 위해 컨트롤러 기준 세이버 위치와 각도를 재조정하고, 판정 범위를 실제 휘두르는 동작에 맞게 수정하였다.

또한 노트가 플레이어 뒤로 지나간 뒤에도 Miss 판정이 명확하지 않은 문제가 있어, 플레이어 좌표 기준으로 노트가 뒤로 지나가면 Miss 처리되도록 기준을 변경하였다.

플레이 중 UI 가시성이 낮은 문제가 있어 HP Bar, Score, Combo HUD의 위치와 크기 및 스타일을 수정하여 가시성을 개선하였다.

## 현재 진행 상태

현재 프로젝트는 VR 리듬 게임의 기본 플레이 흐름이 동작하는 프로토타입 단계이다.

스테이지 선택, 게임 시작, 노트 판정, 점수, 콤보, HP 처리, 게임 종료, 결과 UI 출력, Intro 씬 복귀 흐름까지 구현하였다.

실제 Meta Quest 3S 환경에서 테스트 플레이를 진행하며 노트 속도, 판정 범위, 세이버 위치, UI 가시성 등을 조정하였다.
